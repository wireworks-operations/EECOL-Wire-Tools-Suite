Version History 0.0.1 - 0.6.9

Note: Versions 0.7.1, 0.7.2, 0.7.5, and 0.7.6 are listed above this section.

Early Wire Mark Calculators (0.0.x)

  • v0.0.1: Initial stop mark converter tool created.
  • v0.0.2: Added counting up or down feature because calculations were calculating negative sometimes.
  • v0.0.3: Added marking reference feature to have ability to offset wire marking position.
  • v0.0.5: Added counter to reel distance option, to be able to stop machine precisely when operator exactly wants.
  • v0.0.9: Added clear button for multi cut sessions.

Reel Estimator Development (0.1.x - 0.2.x)

  • v0.1.0: Initial creation of reel estimator.
  • v0.1.2: Added static SVG reel diagram.
  • v0.1.5: Added dynamic coloring to highlighted lines A, B, C, D, E with mouse hover.
  • v0.1.7: Added clearance option for standardized clearance.
  • v0.2.0: Complete reel estimator mathematics calculations overhaul.
  • v0.2.2: Added dynamic SVG reel animation to reel estimation results that show freeboard and layers dynamically according to operator choice.
  • v0.2.5: Initial creation of wire weight estimator.
  • v0.2.7: Overhauled wire weight estimator mathematical calculations.

Knowledgebase and Cutting Records Expansion (0.3.x - 0.4.x)

  • v0.3.0: Created knowledgebase, and advanced math pages for operator and industrial professional knowledgebase.
  • v0.3.3: Created credits pages.
  • v0.4.5: Initial creation of cutting records page.
  • v0.4.6: Added coil or reel selector option for coil or reel type orders.
  • v0.4.7: Added clear all records button and import/export to CSV functions.
  • v0.4.8: Added search options, and search bar for order history.
  • v0.4.9: Added quick stats bar for performance statistics.

Advanced Features and Batch Mode (0.5.x)

  • v0.5.0: Added full pick option.
  • v0.5.1: Added no marks option.
  • v0.5.2: Added system cut option.
  • v0.5.5: Added batch mode for multiple cut orders.
  • v0.5.6: Added start/end marks to batch options.
  • v0.5.7: Added coil/reel option to batch mode.
  • v0.5.8: Added cloud import/export options.
  • v0.5.9: Added print options to all pages results and all wire records.

UI Overhaul and Navigation (0.6.x)
